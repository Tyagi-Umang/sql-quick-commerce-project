# SQL Project: Quick-Commerce Sales, Delivery & Customer Retention Analysis

## Project overview
This project designed to analyze a quick-commerce business such as Blinkit or Zepto using SQL. It focuses on sales performance, delivery efficiency, customer retention, return patterns, and discount analysis.

## Business problem
Quick-commerce companies need fast operational reporting and customer insights to improve revenue, reduce delays, and increase repeat purchases. In this project, I created a small relational database and solved business-focused questions using SQL.

## Objective
The goal of this project is to:
- Analyze order and revenue trends
- Understand customer repeat behavior
- Measure delivery delays
- Identify high-performing categories and products
- Evaluate returns and refund drivers
- Showcase practical SQL skills for a Data Analyst role

## Dataset structure
The database contains 6 tables:
- `customers`
- `products`
- `orders`
- `order_items`
- `deliveries`
- `returns`

## SQL skills demonstrated
- Joins
- Common Table Expressions (CTEs)
- Window Functions
- Aggregate Functions
- CASE WHEN logic
- Date Functions
- Views
- Basic data modeling

## Key business questions answered
1. What are the monthly revenue and order trends?
2. Which categories generate the highest revenue?
3. Who are the top customers by lifetime value?
4. What is the repeat customer rate?
5. Which cities have the highest delivery delay rate?
6. Which acquisition channels bring the best average order value?
7. Which products and categories drive returns and refunds?
8. How much discount is being offered across products?
9. Which customers are active vs at risk of churn?
10. What is the running monthly revenue trend?
11. How much revenue lost due to delivery delays?

## Files in this project
- `schema_and_data.sql` → table creation and sample data insertion
- `analysis_queries.sql` → analyst-style SQL queries and reusable view
- `README.md` → project explanation for portfolio use

## Insights
- Groceries and household items tend to contribute a large share of order value.
- Repeat customers are important for retention-focused growth.
- Delivery delays directly affect refunds and customer experience.
- Acquisition channels can differ in both order volume and average order value.
